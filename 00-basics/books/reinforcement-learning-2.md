## 这本书适合谁

- 有机器学习基础，想进入 sequential decision making 的读者。
- 想理解机器人如何通过交互学习策略的读者。
- 后续打算看模仿学习、offline RL 或 long-horizon planning 的读者。

## 这本书通常用来补什么

- MDP、价值函数、策略改进这些最基础的决策框架。
- TD、MC、Q-learning、policy gradient 等经典方法背后的统一思路。
- 探索、信用分配和长期回报这些在具身任务里反复出现的核心问题。

## 为什么推荐给具身智能学习者

即便很多新工作不再直接写成“纯 RL”，长期回报、延迟奖励和策略优化这套思维仍然是理解具身决策问题的底层工具。

## 建议怎么读

1. 先把 Bellman 视角真正理解，再去看各种算法分支。
2. 不要急着追公式推导，先知道每类方法在解决什么偏差或数据效率问题。
3. 如果你目标是机器人任务，读的时候要一直想“状态、动作、奖励”在真实系统里分别对应什么。

## 按章节怎么学

<a id="rl-ch1"></a>
### 第1章 导论

- 核心问题：强化学习到底和监督学习、规划、控制有什么区别。
- 阅读提示：如果这章没吃透，后面所有算法都会像技巧合集而不是统一框架。

<a id="rl-ch2"></a>
### 第2章 多臂赌博机

- 核心问题：怎样在“现在试试别的”与“继续用当前最好动作”之间平衡。
- 阅读提示：这章虽然简单，但几乎是后面所有探索策略的缩影。

<a id="rl-ch3"></a>
### 第3章 有限马尔可夫决策过程

- 核心问题：把环境形式化成状态、动作、转移和回报。
- 阅读提示：建议把导航或机械臂任务自己翻译成一个 MDP。

<a id="rl-ch4"></a>
### 第4章 动态规划

- 核心问题：已知模型时，如何通过价值迭代和策略迭代求最优策略。
- 阅读提示：重点理解 Bellman 方程，而不是只记具体更新步骤。

<a id="rl-ch5"></a>
### 第5章 蒙特卡洛方法

- 核心问题：没有环境模型时，如何通过完整轨迹回报学习。
- 阅读提示：这章适合和 TD 对照着读，体会偏差和方差的差异。

<a id="rl-ch6"></a>
### 第6章 时序差分学习

- 核心问题：如何边交互边更新，而不用等完整回合结束。
- 阅读提示：先理解 bootstrap 的直觉，再看 Q-learning 这类具体算法。

<a id="rl-ch7"></a>
### 第7章 n步自举法

- 核心问题：如何在短视的 TD 和完整回报的 MC 之间找到中间地带。
- 阅读提示：这是理解后续 GAE、优势估计等现代技巧的好入口。

<a id="rl-ch8"></a>
### 第8章 基于表格型方法的规划和学习

- 核心问题：真实经验与模拟规划怎样组合提升样本效率。
- 阅读提示：读这章时可以开始把它和 model-based 思路联系起来。

<a id="rl-ch9"></a>
### 第9章 基于函数逼近的同轨策略预测

- 核心问题：状态空间一大，表格法为什么不够。
- 阅读提示：这是从经典 RL 走向深度 RL 的第一步。

<a id="rl-ch10"></a>
### 第10章 基于函数逼近的同轨策略控制

- 核心问题：函数逼近进入控制问题后，为什么稳定性突然变难。
- 阅读提示：把近似误差、分布漂移和更新耦合放在一起看。

<a id="rl-ch11"></a>
### 第11章 基于函数逼近的离轨策略方法

- 核心问题：离轨学习如何复用历史数据，又为什么容易发散。
- 阅读提示：这章很适合和 replay buffer、offline RL 联系起来。

<a id="rl-ch12"></a>
### 第12章 资格迹

- 核心问题：如何把长期效果更有效地回传给早期动作。
- 阅读提示：如果你关心长时程任务，这章值得慢一点读。

<a id="rl-ch13"></a>
### 第13章 策略梯度方法

- 核心问题：为什么有时直接优化策略比学值函数更自然。
- 阅读提示：这是理解 actor-critic 路线的关键前置。

<a id="rl-ch14"></a>
### 第14章 心理学

- 核心问题：强化学习框架和动物/人类行为学习之间有哪些映射。
- 阅读提示：这章主要用于拓展视野，不一定要第一遍深挖。

<a id="rl-ch15"></a>
### 第15章 神经科学

- 核心问题：RL 为什么能和奖赏机制、预测误差等脑科学概念对上。
- 阅读提示：把它看成“这套理论为什么有解释力”的补充材料。

<a id="rl-ch16"></a>
### 第16章 应用及案例分析

- 核心问题：经典 RL 在真实问题中怎样落地。
- 阅读提示：建议和机器人或游戏案例一起看，避免停留在抽象算法层。

<a id="rl-ch17"></a>
### 第17章 前沿技术

- 核心问题：强化学习还在往哪些方向扩展。
- 阅读提示：适合作为读完基础后再回来的章节。
