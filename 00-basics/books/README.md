## 建议用法

1. 如果你是新手，先看《机器人学导论》或《现代机器人学》。
2. 如果你来自 AI 背景，再补《Probabilistic Robotics》和 ROS 相关书。
3. 如果你来自机械 / 控制背景，再补《深度学习》和《强化学习（第2版）》。
4. 如果你想快速进入具身项目实践，可以优先看导读页中“具身实践”相关条目。
