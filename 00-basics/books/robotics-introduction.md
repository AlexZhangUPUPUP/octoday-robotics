## 这本书适合谁

- 想系统入门机器人学的初学者。
- 已经会写代码，但对机器人坐标系、运动学、动力学还不扎实的读者。
- 后续准备学习机械臂、控制或 manipulation 的读者。

## 这本书通常用来补什么

- 坐标变换、姿态表示、正逆运动学。
- 速度、雅可比、动力学等机器人学核心基础。
- 轨迹规划与基础控制问题的经典表达方式。

## 为什么推荐给具身智能学习者

只要任务进入真实世界，动作、约束、执行误差和几何关系就绕不过去。这本书的价值在于帮你把“机器人为什么这样动”这件事真正讲清楚。

## 建议怎么读

1. 先把坐标系、齐次变换和正逆运动学读扎实。
2. 再进入雅可比和动力学，不要只记公式，要理解物理意义。
3. 读完后回到仓库里的 [工具与平台](../../04-tools.md#hardware) 看机器人平台，会更容易建立对应关系。

## 按章节怎么学

<a id="craig-ch1"></a>
### 第1章 绪论

- 本章讲什么：从机器人操作臂的应用背景切入，先建立“机器人由机构、驱动、控制组成”的整体认识，再统一全书符号体系。
- 为什么重要：如果一开始不把坐标、关节、末端执行器、任务空间这些词吃透，后面所有公式都会像孤立记号。
- 建议关注：把“机构层”和“控制层”分开理解，先知道机器人在解决什么问题，再进入具体数学。

<a id="craig-ch2"></a>
### 第2章 空间描述和变换

- 本章讲什么：系统介绍位置、姿态、坐标系之间的映射关系，以及平移、旋转、齐次变换等基本工具。
- 为什么重要：这是整本书最底层的语言层。正运动学、逆运动学、视觉标定、控制坐标系都建立在这里。
- 建议关注：不要只背变换矩阵，重点理解“一个向量是在哪个坐标系下表达的”。

<a id="craig-ch3"></a>
### 第3章 操作臂运动学

- 本章讲什么：从连杆描述和坐标系附着规则出发，建立从关节变量到末端位姿的正运动学模型。
- 为什么重要：它回答的是“关节一动，手爪到哪里去”这个最基础的问题，也是后续规划与控制的入口。
- 建议关注：把连杆参数、关节空间、笛卡儿空间之间的关系串起来，避免把参数化和物理结构割裂开。

<a id="craig-ch4"></a>
### 第4章 操作臂逆运动学

- 本章讲什么：讨论给定末端目标位姿时，如何反求一组或多组关节解，包括可解性分析、几何法、代数法和典型结构解法。
- 为什么重要：真实任务通常先给目标抓取位姿，再去找可执行关节角，所以逆运动学是“任务到动作”的第一步。
- 建议关注：重点看多解、无解、奇异附近不稳定这些工程问题，而不只是推公式。

<a id="craig-ch5"></a>
### 第5章 速度和静力

- 本章讲什么：从刚体线速度、角速度出发，推到连杆间速度传播、雅可比矩阵、奇异性和力/速度映射。
- 为什么重要：雅可比是机器人学里最常用的桥梁，它把关节速度和末端运动、关节力矩和末端力联系起来。
- 建议关注：把“雅可比列向量的物理意义”看明白，后面做控制、操控性分析和力控制都会更顺。

<a id="craig-ch6"></a>
### 第6章 操作臂动力学

- 本章讲什么：从加速度、惯量、牛顿-欧拉方程到拉格朗日法，建立关节力矩和运动之间的动力学关系。
- 为什么重要：运动学只能告诉你“怎么动”，动力学才回答“要施加多大力矩才能这样动”。
- 建议关注：优先理解质量矩阵、科里奥利/离心项、重力项这几个结构，而不是一上来陷进繁琐推导。

<a id="craig-ch7"></a>
### 第7章 轨迹的生成

- 本章讲什么：区分路径与轨迹，介绍关节空间规划、笛卡儿空间规划、实时轨迹生成和无碰撞路径规划。
- 为什么重要：机器人最终执行的不是单个点位，而是一条随时间演化的可执行运动。
- 建议关注：把“几何路径”和“时间参数化”分开看，这对理解后续运动规划系统很关键。

<a id="craig-ch8"></a>
### 第8章 操作臂的机械设计

- 本章讲什么：从任务需求出发看机构构型、工作空间、冗余与闭链、驱动方式、刚度、位置与力传感。
- 为什么重要：很多控制和算法问题，根子其实在机构设计。设计错了，后面软件再强也补不回来。
- 建议关注：把工作空间、可达性、刚度和传感这几项放到具体任务里想，而不是抽象地看指标。

<a id="craig-ch9"></a>
### 第9章 操作臂的线性控制

- 本章讲什么：围绕反馈闭环、二阶系统、轨迹跟踪、抗干扰和工业控制器结构建立线性控制直觉。
- 为什么重要：这是从“建立模型”进入“让机器人稳定执行”的第一章，也是理解工业机器人控制器的基础。
- 建议关注：把误差动力学和控制律分解吃透，理解 PID、前馈和轨迹跟踪为什么能工作。

<a id="craig-ch10"></a>
### 第10章 操作臂的非线性控制

- 本章讲什么：进一步讨论非线性、时变、多输入多输出系统，以及李雅普诺夫分析、笛卡儿空间控制和自适应控制。
- 为什么重要：机器人本体天然是强非线性的，很多在线性近似下表现良好的方法，到了大范围运动就会失真。
- 建议关注：把“为什么要做非线性补偿”和“稳定性如何证明”两件事分开理解。

<a id="craig-ch11"></a>
### 第11章 操作臂的力控制

- 本章讲什么：从装配任务和局部约束任务出发，讨论力控制、力/位混合控制和接触环境中的控制坐标系。
- 为什么重要：具身系统进入真实物理环境后，不只是到达某个位姿，更要安全、稳定地和环境交互。
- 建议关注：理解哪些方向应该控位置、哪些方向应该控力，这是接触任务设计的核心。

<a id="craig-ch12"></a>
### 第12章 机器人编程语言及编程系统

- 本章讲什么：介绍机器人编程的不同层级、语言应具备的能力、典型应用和编程系统的约束。
- 为什么重要：算法只有进入编程系统，才能真正变成可部署的机器人行为。
- 建议关注：把“动作层指令”“任务层描述”“系统级调度”之间的差别看清楚。

<a id="craig-ch13"></a>
### 第13章 离线编程系统

- 本章讲什么：讨论仿真驱动的离线编程系统，包括核心问题、引导仿真器和自动化子任务。
- 为什么重要：工业场景里很多程序不能直接在真机上反复试错，离线环境就是降低成本和风险的关键。
- 建议关注：把它当成今天数字孪生、仿真验证和自动部署流程的早期框架来看，会更容易和现代工具对应起来。

## 建议阅读顺序

- 如果你是第一次学机器人，优先读第 2 到第 5 章，再补第 6、7、9 章。
- 如果你更偏机械臂控制，主线是第 3、5、6、9、10、11 章。
- 如果你更偏工程落地，可以把第 8、12、13 章和仓库里的 [工具与平台](../../04-tools.md) 一起看。
