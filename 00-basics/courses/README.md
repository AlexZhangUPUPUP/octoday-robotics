## 建议用法

1. 如果你需要系统机器人学基础，先看 CS223A。
2. 如果你需要深度学习基础，先看 MIT 6.S191。
3. 如果你想补强化学习，可先看《强化学习（第2版）》再进入 CS285 或《动手学强化学习》。
4. 如果你目标是 manipulation，可重点看 MIT 6.4210。
