- 课程主页：[https://introtodeeplearning.com/](https://introtodeeplearning.com/)

- 备注：官方页明确说明这是高强度入门项目，覆盖 vision、language、robotics 等应用。

## 这门课适合谁

- 想快速补全深度学习基本概念与实战方法的读者。
- 还没准备好直接读具身大模型论文的读者。

## 这门课通常用来补什么

- 神经网络、训练流程、生成模型和前沿应用的紧凑式入门。
- 面向实践的 DL 基础框架。

## 为什么推荐给具身智能学习者

作为进入视觉、多模态和具身基础模型之前的热身课程，这门课效率很高。

## 建议怎么学

1. 如果你是零基础，先过这门课再回到《深度学习》。
2. 适合在短时间内快速补齐背景。

## 课程导航

以下表格按 MIT 6.S191 2026 官方 schedule 整理。

| 讲次 | 主题 | 学习重点 |
| :--- | :--- | :--- |
| Lecture 1 | Intro to Deep Learning | 建立神经网络训练和表示学习的最小骨架 |
| Lecture 2 | Deep Sequence Modeling | 把序列建模、语言建模和时序输入放进统一视角 |
| Lecture 3 | Deep Computer Vision | 先把视觉任务和卷积网络主线理顺 |
| Lecture 4 | Deep Generative Modeling | 理解生成模型为什么不仅是“会生成图像” |
| Lecture 5 | Deep Reinforcement Learning | 把监督学习之外的决策学习纳入你的知识图谱 |
| Lecture 6 | New Frontiers | 快速浏览当前前沿方向和系统趋势 |
| Lecture 7 | AI for Science | 理解 AI 在科学问题中的落地方向 |
| Lecture 8 | Secrets to Massively Parallel Training | 知道大规模训练为什么会成为系统工程问题 |
| Lecture 9 | The Three Laws of AI | 回到部署、责任和系统边界这些问题 |

## 实践环节

官方页面同时公开了 software labs 与 final project 节点。

| 环节 | 主题 | 学习重点 |
| :--- | :--- | :--- |
| Lab 1 | Deep Learning in Python; Music Generation | 先把训练流程和代码环境真正跑起来 |
| Lab 2 | Facial Detection Systems | 把视觉模型落到一个完整任务上 |
| Lab 3 | Fine-Tune an LLM, You Must! | 体验微调流程和参数更新带来的行为变化 |
| Project Stage 1 | Final Project | 练习把课程知识转成一个可执行问题 |
| Project Stage 2 | Project Work | 在开放题里验证自己是否真正理解了方法 |
| Project Stage 3 | Project Presentations | 学会把问题定义、方法选择和结果表达说清楚 |
