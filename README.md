# 🤖 星期八 · 具身智能开源社区

## 🌟 我们的愿景

> 🚀 一个连接「产业 × 资本 × 学生 × 人才」的具身智能开源社区首页
> An open community for learners, researchers, builders, and practitioners across Embodied AI。

**“星期八”意味着额外的一天——我们希望在这个领域，为你提供超越常规的资源和视角。**

这里不是又一个论文列表，而是一个**连接产业、人才与知识**的信息枢纽。无论你是刚踏入具身智能领域的学生，还是寻找落地场景的工程师，或是关注行业动态的投资人，都能在这里找到自己需要的那块拼图。

## 🪜 内容导航

> 如果你是第一次来到这个社区，可以按下面的顺序浏览。它更像使用建议，而不是固定学习路径。

### 1.1 先建立基础认知框架

从 [基础知识总览](00-basics.md) 开始：

- 先看 [推荐书籍](00-basics.md#recommended-books)，适合补齐机器人学、强化学习、视觉语言模型、控制等核心概念。
- 再看 [在线课程](00-basics.md#online-courses)，适合建立从感知、规划、控制到学习的系统理解。
- 如果你刚接触这个方向，建议先把基础部分快速过一遍，再进入论文、产业和岗位板块。

### 1.2 再按研究主题进入论文与开源代码

进入 [论文&代码总览](03-papers-code.md)，按兴趣选择方向：

- [Embodied Foundation Models](03-papers-code.md#embodied-foundation-models)：适合先看“通用机器人智能底座”正在怎么做。
- [Vision-Language-Action (VLA)](03-papers-code.md#vision-language-action-vla)：适合关注当前最热的视觉-语言-动作闭环方向。
- [Embodied Agents & Reasoning](03-papers-code.md#embodied-agents-reasoning)：适合看任务规划、长时程决策和 Agent 化能力。
- [Manipulation](03-papers-code.md#manipulation)：适合看机械臂、人形手部、灵巧操作与抓取。
- [Navigation & Spatial Intelligence](03-papers-code.md#navigation-spatial-intelligence)：适合看导航、空间理解、世界建模。
- [Simulation & Sim2Real](03-papers-code.md#simulation-sim2real)：适合看仿真训练如何过渡到真机部署。
- [Datasets](03-papers-code.md#datasets)：适合找训练数据、数据采集方案与公开资源。
- [Benchmarks & Evaluation](03-papers-code.md#benchmarks-evaluation)：适合看领域里怎么评测模型和系统能力。
- [Survey](03-papers-code.md#survey)：适合想快速扫全景、建立研究地图的人。

### 1.3 用工具与平台补足工程视角

结合 [工具与平台](04-tools.md) 一起看，会更容易把研究方向映射到工程实践：

- [仿真平台与物理引擎](04-tools.md#simulation)：适合找训练、验证和场景复现环境。
- [具身智能平台与开发框架](04-tools.md#framework)：适合找开发框架、系统底座和常用平台。
- [机器人硬件与开发平台](04-tools.md#hardware)：适合找机器人本体、开发板和试验平台。
- [机器人开发工具与 SDK](04-tools.md#devtools)：适合找调试、控制和工程化工具链。
- [机器人公司开源项目](04-tools.md#opensource)：适合按公司维度看谁在持续输出开源能力。

### 1.4 单独查看公司列表，理解产业版图

进入 [公司列表总览](01-companies.md)：

- [国内公司](01-companies.md#china-companies)：适合梳理中国本土整机厂商、核心零部件企业、平台公司与典型应用方向。
- [国外公司](01-companies.md#global-companies)：适合观察海外头部实验室、机器人公司、芯片企业和工具链玩家。
- 公司表格里已经包含公司简介、标签和官网，适合做产业扫描、竞品研究、投资梳理和求职目标筛选。

### 1.5 单独查看招聘机会，理解人才需求

进入 [招聘信息总览](02-jobs.md#jobs-overview)：

- [国内机会](02-jobs.md#jobs-domestic)：适合关注国内机器人公司、平台公司、研究机构的全职、校招和实习机会。
- [海外机会](02-jobs.md#jobs-overseas)：适合关注海外公司、大学实验室和研究团队岗位。
- [专项人才与顶尖计划](02-jobs.md#jobs-special)：适合查找博士后、人才计划、顶尖专项和特殊培养项目。
- 招聘页里每家公司都已经拆成独立表格，更方便横向比较岗位方向、地点、类别和投递入口。

### 1.6 最后看贡献方式，参与社区共建

如果你想参与共建、投稿或推荐资源，可以从这里开始：

- [贡献指南](CONTRIBUTING.md)：了解补充资源、修正文案、提交 PR 的基本方式。
- **邮箱**：octoday@yeah.net
- **GitHub Issues**：直接在本仓库提交 [Issue](https://github.com/AlexZhangUPUPUP/octoday-robotics/issues)
