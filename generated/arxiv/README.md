# Generated arXiv Snapshots

This directory stores generated embodied-AI arXiv snapshots.

- `daily/latest.md`: latest daily markdown snapshot
- `daily/latest.json`: latest daily JSON snapshot
- `daily/candidates-for-03-papers.md`: daily merge candidates for `03-papers.md`
- `weekly/latest.md`: latest weekly markdown snapshot
- `weekly/latest.json`: latest weekly JSON snapshot
- `weekly/candidates-for-03-papers.md`: weekly merge candidates for `03-papers.md`

Generation scripts:

- `scripts/fetch_embodied_arxiv.py`
- `scripts/update_embodied_arxiv_outputs.py`
