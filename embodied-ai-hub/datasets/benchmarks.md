# Benchmarks & Evaluation

## 重点基准

| Benchmark | 方向 | 看什么 |
| --- | --- | --- |
| ManiBench | 操作 | 细粒度 manipulation 能力 |
| CALVIN | 语言条件策略 | 长时程任务与语言条件控制 |
| RLBench | 机器人学习 | 经典标准任务集 |
| CRAM | 组合推理与操作 | reasoning + action |
| VIMABench | VLA | 视觉语言动作统一评测 |
| OpenEQA | Embodied QA | 具身问答和场景理解 |
| EmbSpatial-Bench | 空间推理 | embodied spatial reasoning |
| BEHAVIOR-1K | 日常活动 | 大规模 everyday tasks |
| Franka Kitchen | 长时程操作 | 长 horizon manipulation |
| Habitat Challenge | 导航 | 3D 场景下导航评测 |

## 看 benchmark 时最该关注什么

1. 评测协议是不是公开清晰。
2. 任务是否真的接近真实部署条件。
3. 是否有配套代码、数据与复现路径。
