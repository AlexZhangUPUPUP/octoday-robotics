## 投递前建议

1. 简历里不要只写“大模型”，要写清你在 `仿真 / 真机 / 数据 / 控制 / 评测` 中具体做了什么。
2. 如果投算法岗，至少要有一段可复现的 `RL / IL / VLA` 项目经历。
3. 如果投系统工程岗，尽量给出 `ROS2 + 部署 + 调试 + 硬件联调` 的实战证据。
4. 如果投研究岗，论文理解深度、实验复现和 benchmark 意识比“看过很多文章”更重要。
