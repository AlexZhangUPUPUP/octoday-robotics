# Simulation & Sim2Real

## 代表工作

| 论文 / 平台 | 时间 | 价值 |
| --- | --- | --- |
| MuJoCo | 2012 | 控制与 RL 经典物理引擎 |
| AI2-THOR | 2017 | 交互式 3D 视觉环境 |
| RoboSuite | 2020 | 模块化机器人学习框架 |
| SAPIEN | 2020 | 部件级交互仿真 |
| Isaac Gym | 2021 | GPU 高性能仿真 |
| OmniGibson | 2023 | 模块化具身环境 |
| RoboCasa | 2024 | 大规模日常任务仿真 |
| Genesis | 2024 | 通用生成式物理引擎 |
| UniSim | 2025 | 通用仿真器路线 |
| Sim2Real 2.0 | 2025 | Sim2Real 综述与基准 |

## 迁移路线里值得看什么

- 视觉编码器预训练是否能缩小现实差距。
- `real-to-sim-to-real` 是否建立了更稳健的数据闭环。
- 零样本 sim2real 是否真的适用于复杂任务，而不是单一 benchmark。
