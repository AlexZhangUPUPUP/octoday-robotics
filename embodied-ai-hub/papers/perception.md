# Perception & Spatial Intelligence

这一页聚焦具身感知、导航和空间推理。

## 推荐论文

| 论文 | 关键词 | 为什么值得看 |
| --- | --- | --- |
| NavGPT | Vision-Language Navigation | 导航中的显式推理 |
| VLFM | Zero-shot semantic navigation | 视觉语言地图与零样本导航 |
| CoW | Chain-of-Thought Walking | 把 CoT 引入导航决策 |
| RoboTracer | Spatial trace reasoning | 机器人场景中的空间轨迹推理 |
| RoboRefer | Spatial referring | 视觉语言中的空间指代理解 |
| CLIP-Fields | Robotic memory | 场景记忆与语义场表示 |
| Active Neural SLAM | Exploration + memory | 神经 SLAM 代表作 |
| Habitat | Embodied research platform | 导航与 3D 场景研究基础平台 |
| HM3D | 3D environments | 导航常用大规模 3D 环境数据集 |
| EgoVLPv2 | Egocentric video-language | 第一人称视角感知的重要路线 |

## 这条线最值得一起看什么

- 数据集：`HM3D`, `EmbodiedScan`, `Ego4D`, `ScanNet`
- 基准：`OpenEQA`, `EmbSpatial-Bench`, `Habitat Challenge`
- 工具链：`Habitat Sim`, `Habitat-Lab`, `SAPIEN`, `Genesis`

## 建议阅读顺序

1. 从 `Habitat`、`HM3D` 了解导航研究基础设施。
2. 再读 `NavGPT`、`VLFM`、`CoW`，补视觉语言导航主线。
3. 最后读 `RoboTracer`、`RoboRefer`、`CLIP-Fields`，补空间推理与记忆。
