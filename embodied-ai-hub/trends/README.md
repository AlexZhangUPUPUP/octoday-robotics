# 趋势雷达

这个版块不追求“今天又多了几篇 arXiv”，而是帮助你建立一套长期有效的观察框架。

## 重点跟踪对象

| 类别 | 建议重点关注 |
| --- | --- |
| 顶会 | CoRL, RSS, ICRA, IROS, CVPR, NeurIPS |
| 研究机构 | Stanford, UC Berkeley, MIT, CMU, Tsinghua, DeepMind Robotics, Meta FAIR |
| 公司方向 | Figure AI, Physical Intelligence, Apptronik, 1X, Unitree, AgiBot, Galaxy, Deep Robotics |
| 技术路线 | VLA, Embodied Foundation Models, World Models, Manipulation, Sim2Real, Spatial Reasoning |
| 工程底座 | Isaac Sim, MuJoCo, Habitat, ROS2, 数据采集与评测体系 |

## 最近几年最清晰的几条主线

1. 从单任务策略走向通用机器人基础模型。
2. 从感知-规划-控制分层走向更强的端到端 VLA 与 agentic reasoning。
3. 从“仿真够不够真”转向“数据闭环和评测闭环是否足够强”。
4. 从实验室 demo 转向真实部署、量产和场景 ROI。

## 观察一个方向是否真的在升温

| 观察信号 | 为什么有用 |
| --- | --- |
| 同一主题在顶会、产业和招聘里同时出现 | 说明不只是论文热度，而是开始形成生态 |
| 数据集、benchmark、工具链一起出现 | 说明方向开始具备复现土壤 |
| 公司开始发布岗位、产品和 demo 闭环 | 说明离工程落地更近 |
| 综述、benchmark 和 survey 变多 | 说明方向进入系统化梳理阶段 |
