# 学习路线图

## 7 阶段路径

| 阶段 | 重点 | 推荐动作 |
| --- | --- | --- |
| 阶段 1：夯实基础 | 机器人学、深度学习、强化学习 | 先补书和基础课 |
| 阶段 2：核心技能 | 概率机器人学、RL、ROS | 跑通第一个机器人软件栈 |
| 阶段 3：实践仿真 | MuJoCo / Isaac Sim / SAPIEN | 在仿真里复现基线 |
| 阶段 4：前沿探索 | VLA、Diffusion Policy、World Model | 建立当前主流研究图谱 |
| 阶段 5：动手项目 | 数据收集、训练、评测、部署 | 至少做一个端到端项目 |
| 阶段 6：保持跟踪 | 顶会、arXiv、实验室更新 | 建立长期信息流 |
| 阶段 7：进阶深化 | 灵巧操作、人形控制、具身大模型 | 聚焦一条线深挖 2-3 年论文 |

## 阶段拆解

### 阶段 1：夯实基础

- 读《机器人学导论》或《现代机器人学》。
- 完成 MIT 6.S191 的前半段。
- 理解 RL 的基本概念和常见算法族。

### 阶段 2：核心技能

- 学 `CS285`，补 PPO、SAC 等主流算法。
- 学 `Probabilistic Robotics` 前几章。
- 用 `ROS2 + Gazebo` 跑通最小机器人项目。

### 阶段 3：实践仿真

- 选一个仿真器，不要同时开很多坑。
- 在 `MuJoCo / Isaac Sim / SAPIEN` 中至少复现一个经典任务。

### 阶段 4：前沿探索

- 读 `RT-2`, `OpenVLA`, `π0`, `Diffusion Policy`, `DreamerV3`。
- 建议同步浏览 [papers](../papers/README.md)。

### 阶段 5：动手项目

- 做一条完整链路：数据 -> 训练 -> 评测 -> 仿真/真机。
- 能进开源社区就尽量提交 issue 或 PR。

### 阶段 6-7：持续深化

- 建立每周固定跟踪节奏。
- 最终选一条主线深挖：`manipulation`, `navigation`, `VLA`, `humanoid control`。
