# 推荐书单

这一页把原始书单重新整理成更适合按目的阅读的版本。

## 核心书单

| 书名 | 适合谁 | 你会得到什么 |
| --- | --- | --- |
| 《机器人学导论》 | 初学者 | 运动学、动力学、轨迹规划入门 |
| 《现代机器人学》 | 理论导向读者 | 更现代的机器人学表达与数学体系 |
| 《强化学习（第 2 版）》 | AI 背景学习者 | RL 基础范式和经典算法 |
| 《深度学习》 | 通用 AI 学习者 | 视觉、语言、感知模块的基础 |
| 《Probabilistic Robotics》 | 感知/定位研究者 | SLAM、状态估计、不确定性建模 |
| 《Robot Learning》 | 机器人学习方向 | 模仿学习、强化学习、元学习主线 |
| 《Embodied AI: How to Train Your Robot》 | 工程实践者 | 从环境搭建到策略部署的实操视角 |
| 《The Robot Operating System for Absolute Beginners》 | 开发者 | ROS/ROS2 的软件工程入门 |
| 《Manipulation: The Mechanics of Robotic Grasping and Manipulation》 | 操作方向研究者 | 抓取与操作力学基础 |
| 《Robotics, Vision and Control》 | 原型验证型读者 | 机器人、视觉、控制一体化实践 |

## 按目标选书

| 目标 | 建议先读 |
| --- | --- |
| 建立机器人学底座 | 《机器人学导论》, 《现代机器人学》 |
| 补 AI 与学习能力 | 《强化学习》, 《深度学习》 |
| 走向真实机器人系统 | 《ROS for Absolute Beginners》, 《Probabilistic Robotics》 |
| 深挖 manipulation | 《Manipulation》, 《Robot Learning》 |
| 想快点动手 | 《Embodied AI: How to Train Your Robot》 |

## 推荐阅读顺序

1. 先读机器人学，再读强化学习和深度学习。
2. 如果偏工程，尽早补 ROS 与仿真环境。
3. 如果偏研究，再深入概率机器人学、操作和机器人学习。
