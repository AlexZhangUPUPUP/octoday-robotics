# 术语速查

## 高频术语

| 术语 | 一句话解释 |
| --- | --- |
| Embodied AI | 智能体通过身体与环境闭环交互形成能力 |
| Sim2Real | 策略从仿真迁移到真实世界 |
| VLA | 视觉-语言-动作统一模型 |
| World Model | 用于预测、规划和想象的内部环境模型 |
| Diffusion Policy | 基于扩散模型的动作生成方法 |
| Imitation Learning | 模仿专家示教学习策略 |
| Reinforcement Learning | 基于奖励信号优化策略 |
| PPC | 感知-规划-控制三层机器人架构 |
| Scene Representation | 机器人内部对环境的结构化表达 |
| Manipulation | 抓取、放置、组装等精细操作任务 |
| Locomotion | 双足、四足、轮式等移动能力 |
| Teleoperation | 人类远程操控机器人 |
| Cross-Embodiment Transfer | 不同机器人形态之间的策略迁移 |
| Point Cloud | 三维点集合，常用于空间感知 |
| Task Planning | 高层任务分解与步骤安排 |
| Skill Learning | 学习可复用的原子技能 |
| Dataset & Benchmark | 数据资源与标准化评测任务 |

## 建议怎么用

- 看论文前先扫一遍，把陌生词打平。
- 做项目时优先明确这些词在自己系统里分别落在哪一层。
