# 硬件平台

## 代表平台

| 平台 | 方向 | 特点 |
| --- | --- | --- |
| TurtleBot 4 | 教学 / ROS2 | 开箱即用，适合入门 |
| Clearpath Husky | 科研底盘 | 户外移动机器人标准平台 |
| Scout 系列 | 移动底盘 | 室外非结构地形适配 |
| myAGV | 桌面级移动平台 | 开源、低门槛 |
| kscale K-Bot | 开源人形平台 | 面向开发者研究 |
| Unitree Go2 / H1 / G1 | 四足 / 人形 | 控制生态和硬件能力强 |
| Figure 02 | 通用人形 | 端到端学习与工业场景 |
| 1X NEO | 家庭服务人形 | 家庭场景定位鲜明 |
| Sanctuary Phoenix | 人形 | 强调灵巧手和触觉 |

## 选择建议

- 教学和软件栈学习优先 `TurtleBot 4`。
- 做移动机器人研究看 `Husky / Scout`。
- 做具身和人形方向再看 `Unitree / K-Bot / Figure`。
