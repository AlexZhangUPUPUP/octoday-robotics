## 看公司时最值得问的四个问题

1. 它卖的是整机、平台、模型，还是核心部件？
2. 它的优势来自算法、控制、硬件、数据，还是系统集成？
3. 它的场景是真的高频刚需，还是偏展示性？
4. 它是否有自己的数据闭环、部署闭环和迭代闭环？
