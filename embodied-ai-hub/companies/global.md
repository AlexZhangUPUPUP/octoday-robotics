# 海外公司观察

## 代表玩家

| 公司 | 方向 | 观察点 |
| --- | --- | --- |
| Figure AI | 人形、物流制造、VLA | 资本密度和工程推进速度 |
| 1X | 家庭服务人形 | 家庭场景切口是否成立 |
| Apptronik | 人形、工业制造 | Apollo 的工业路线 |
| Agility Robotics | 双足、仓储物流 | Digit 的场景聚焦和商业节奏 |
| Boston Dynamics | 动力学、控制、整机 | 长周期工程能力标杆 |
| Sanctuary AI | 灵巧手、人形 | 操作与触觉能力 |
| Physical Intelligence | 物理世界基础模型 | 大脑层叙事代表玩家 |
| Skild AI | 通用机器人“大脑” | 统一控制不同机器人本体 |
| Intrinsic | 机器人软件平台 | 软件定义机器人路线 |
| Dexterity | 灵活操作、物流自动化 | 非结构化环境中的精细操作 |

## 海外路线特点

- 更强调 `foundation model` 与通用控制。
- 更常把场景聚焦在 `物流 / 制造 / 家庭服务`。
- 软件平台公司和整机公司边界更清晰。
